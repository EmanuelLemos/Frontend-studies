Cores HTML

HTML é a sigla para a expressão inglesa HyperText Markup Language, que significa, em tradução livre para a língua portuguesa, Linguagem de Marcação de Hipertexto. A linguagem é essencialmente utilizada para produzir páginas para web, especificando, por exemplo, a posição e tamanho das imagens; o tipo, cor e tamanho da fonte; a cor de fundo; o tamanho das margens, dentre outros. Ao ler um arquivo HTML, o navegador é capaz de reproduzir no computador local exatamente a página que está em outro computador.

Quanto à utilização de cores, com os códigos HTML podemos definir a cor de fundo do website (background), a cor do texto, células em tabelas, dentre outros. O uso de cores torna o processo de comunicação mais eficiente -- auxilia a visualização de links, possibilita indicar mecanismos de segurança e deixa os terminais de vídeo (monitores, projetores) mais agradáveis.

É importante lembrar que as cores que podemos ver na tela do monitor dependem da interpretação do sistema -- que inclui o navegador (browser), o sistema operacional e alguns recursos do próprio monitor ou projetor (configuração e calibragem). A visualização mais correta das cores só pode ocorrer em monitores configurados a partir de Color (16 bits), True Color (24 bits) ou em modo superior.

Os comprimentos de onda vermelho, verde e azul constituem a base para todas as demais cores da natureza, e por isso essas três cores são chamadas de primárias. Todas as outras cores do espectro são criadas pela combinação de diferentes intensidades desses três comprimentos. O padrão RGB (Red, Green e Blue) é que codifica as misturas cromáticas. Entretanto, as cores podem ser codificadas em diferentes bases.

O valor de cada componente do padrão RGB deve ser um número inteiro variável entre 0 e 255, gerando um código numérico de base 10. A cor vermelha, por exemplo, é representada por #255 0 0, o que significa que ela, possui intensidade máxima de vermelho e mínima de verde e azul. Essa notação é chamada de decimal, entretanto ela é evitada, pois a representação numérica fica extensa. Como alternativa, convencionou-se utilizar a representação hexadecimal.

É importante notar que cada código HTML em sistema numérico hexadecimal é precedido pelo símbolo “#”. Seis letras ou números completam a representação.
